# Aluraplus-clone-interface 📺

Projeto desenvolvido durante o curso da Alura para praticar conceitos fundamentais de HTML5 e CSS3. O site é uma página de destino (landing page) focada em subscrições de serviços de streaming.

## 🚀 Tecnologias Utilizadas

* HTML5: Estruturação semântica do conteúdo.
* CSS3: Estilização, utilização de variáveis e Flexbox/Grid.
* Design Responsivo: Adaptação do layout para dispositivos móveis através de Media Queries.
* GitHub: Controlo de versão e armazenamento do código.
* Vercel: Deploy e hospedagem do site na nuvem.

## 📱 Funcionalidades

* [x] Layout fiel ao design original.
* [x] Botões interativos com efeitos de hover.
* [x] Totalmente responsivo (ajustado para telemóveis).
